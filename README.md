# 🎭 当代人生存人格测试

一个有趣的在线人格测试，通过20道问题测出你在当代社会的真实生存人格。

## 特点

- 📱 响应式设计，手机/平板/电脑均可访问
- 🎨 暗色主题，视觉效果出众
- ⚡ 纯前端实现，无需后端服务器
- 🔄 自动播放动画和交互反馈
- 📊 四维人格分析：执行力、社交力、松弛感、叛逆值

## 8种人格类型

| 类型 | Emoji | 特征 |
|------|-------|------|
| 永动机型打工人 | ⚡ | 效率至上，永远在线 |
| 精致利己主义者 | 🎯 | 精于算计，目标明确 |
| 人间清醒型选手 | 🧠 | 看透一切，理性平衡 |
| 社交恐怖分子 | 🎪 | 社交能量爆棚 |
| 佛系摸鱼大师 | 🐟 | 可持续发展型选手 |
| 精神内耗型哲学家 | 🌪️ | 思考过度，内心戏丰富 |
| 躺平艺术家 | 🛋️ | 躺出水平，躺出境界 |
| 混沌型自由灵魂 | 🌊 | 无拘无束，随性而为 |

直接打开 `index.html` 即可，或通过 GitHub Pages 访问。
